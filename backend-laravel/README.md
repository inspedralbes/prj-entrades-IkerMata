# API Laravel (TICKET-FAST)

Aquest directori conté l’aplicació **Laravel** (API REST, Sanctum, lògica de reserves i compres).

- **Instal·lació i entorn Docker:** vegeu el manual a l’arrel del repositori: [../doc/MANUAL-INSTALACIO-I-CONFIGURACIO.md](../doc/MANUAL-INSTALACIO-I-CONFIGURACIO.md)

La documentació genèrica del framework Laravel es troba a [https://laravel.com/docs](https://laravel.com/docs).
