# Carpeta `specs/` — lliurament (enunciat) + OpenSpec

L’especificació **canònica** del canvi viu sota OpenSpec:

`openspec/changes/sincronitzacio-estat-butaques-inicial/`

Per complir l’enunciat que demana fitxers amb noms concrets (`foundations.md`, `spec.md`, `plan.md`), aquest directori conté **còpies alineades** amb aquell canvi (mateix contingut o resum explícit). Qualsevol actualització futura hauria de fer-se primer a `openspec/changes/...` i després sincronitzar aquí si cal.
